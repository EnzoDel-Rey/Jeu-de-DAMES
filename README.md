# Jeu de Dames Java

Projet réalisé dans le cadre du module de programmation Java.

## Objectif

Développer un jeu de dames jouable à deux joueurs en utilisant Java et la programmation orientée objet.

## Technologies utilisées

Java
Swing pour l'interface graphique

## Justification du choix de Swing

Swing a été choisi car :

- intégré directement dans Java
- facile à utiliser pour un projet débutant
- très documenté
- adapté pour les interfaces simples

JavaFX a été étudié mais jugé plus complexe pour un projet de première année.

## Installation

1 Télécharger le projet

2 Compiler

javac Main.java

3 Lancer

java Main

## Fonctionnalités

- Plateau de dames 8x8
- Affichage des pièces
- Structure POO
- Interface graphique Swing
